HTML Headings
Tag Name Example Description
<h1> H1 Heading

This is an h1 heading

 h1 headings are used to signify the most important heading on the page and commonly used by text-readers
<h2> H2 Heading

This is an h2 heading

 h2 headings are used to represent the 2nd most important headers on the page
<h3> H3 Heading

This is an h3 heading

 h3 headings are used to represent the 3rd most important headers on the page
<h4> H4 Heading

This is an h4 heading

 h4 headings are used to represent the 4th most important headers on the page
<h5> H5 Heading
This is an h5 heading
 h5 headings are used to represent the 5th most important headers on the page
<h6> H6 Heading
This is an h6 heading
 h6 headings are used to represent the least most important headers on the page

HTML Text Formatting
Tag Name Example Description
<b> Bold text This text is bold defines bold text without any extra importance
<strong> Important text This text is strong defines text with strong importance and content is typically displayed in bold
<i> Italic text This text is Italic defines part of the text in an alternate voice or mood and often used to indicate techical terms, text in another language, a thought or name
<em> Emphasized text This text is Emphasized defines parts of text that are Emphasized and are generally read using verbal stress with screen readers
<mark> Marked text This text is marked defines text to be marked or highlighted
<small> Smaller text This is some smaller text defines smaller text
<del> Deleted text This text is deleted defines text that has been deleted and will typically produce text with a strike-through
<ins> Inserted text This text was inserted defines text that has been inserted and browsers will typically underline this text
<sub> Subscript text This is Subscript text text appears half a character below the normal line and sometimes renders in smaller font. Can be used for h2O
<sup> Superscript text This is Superscript text text that appears half a character above the normal line and sometimes renders in smaller font. Can be used for footnotes, like WW1[1]
